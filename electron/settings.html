Release notes for 4.5.3

MTGATracker had TONS of support from Open Source contributors this cycle. Thanks so much to everyone who put some code in!!

  • More actions are now recorded in the Action Log, including:
    • Cards being moved to the graveyard or to exile
    • Lands played for turn
    • Card draws
    • Player's opening hand and any mulligans taken
  • Fixed the main font--yes, this is how MTGATracker was always supposed to look.
  • Fixed split-card tracking (thanks WizardRubic!)
  • Added new per-deck, and per-session win counters (thanks to the excellent & hard work by Khoross and smkiewel!)
  • Added a few new card-sorting methods for decklists (thanks smkiewel!)
  • Decks will now be sorted alphabetically in the main MTGATracker interface (thanks smkiewel!)
  • Added one of GabySpartz' precon deck names (thanks WizardRubic!)
  • MTGATracker can now optionally minimize to the OS tray (again, thanks to the work done by WizardRubic!)
  • Small UI continuity/display fix for the settings button

Release notes for 4.5.1 (4.5.0 hotfix)

  • Fixed an issue where settings files that were generated while KLD/AER/AKH/HOU were in MTGA could cause MTGATracker to fail to operate.
  • Fixed a typo in the settings panel ("Glearly" someone should proofread a little better)
  • Fixed an issue where clearing the action log while the action log window is not open could cause an error dialog

Release notes for 4.5.0

  • A new feature appears! 4.5.0 includes a brand new Game Action History Log! This log contains the following action events:
    • A spell or ability resolves
    • A spell or ability targtes something
    • A creature is attacking or is being blocked
    • A player's life changes
    Armed with these pieces of info, just about any game interaction should be retroactively understandable. Gone are the days of looking away and missing an important event only to be left puzzling how your opponent cleared your board with only three mana up. (It was probably Deafening Clarion, you're welcome.)



    Even better, these logs will also be sent to Inspector, so that you always have a record of exciting games! It's not quite a replay, but it's close!

    To open the Game History window, click the new History icon from the main UI:



  • Fixed an issue that would cause an alert dialog when closing the tracker
